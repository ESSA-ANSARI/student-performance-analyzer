# Student Performance Analyzer

## 📌 Project Overview
This project analyzes student performance using Python.

## 🚀 Features
- Reads data from CSV file
- Calculates average marks
- Assigns grades
- Finds topper and lowest scorer

## 🛠️ Technologies Used
- Python
- File Handling

## 📊 Sample Output
Alice → Avg: 81.8 → Grade: A  
Sara → Avg: 84.8 → Grade: A  
John → Avg: 76.6 → Grade: B+  

Final Result:  
Topper: Melisa (92.8)  
Lowest: Madison (73.2)
